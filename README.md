# byte

Byte Language compiler, libraries, and tools.

## About

Byte is a statically and strongly-typed language with a strong focus on 
meta-programming.

The language goal is to be accessible and easy to use, while still providing
advanced features and high performance when necessary. Another goal is to be
available for any platform, be it native executables, or running on the web
transpiled to JavaScript.

## Architecture

In its core, Byte is a declarative language that runs at compile time to produce
the AST of the actual program that will be compiled and executed at run time.

This full on meta-programming approach coupled with a powerful type system
provides the ultimate flexibility to address any problem.
